TBVx_targets_template.csv

The calibration targets for the country of interest.

Overall Notes: 
- Only targets for ONE country specified in one csv file
- There are 5 rows specifying 5 different types of calibration targets
- Multiple years and/or age groups can be included for the same type of calibration targets


Notes for specific columns:

country
- 3 letter country code for the country of interest

year
- The year of the target + 0.5 (for calibration)

err
- (lo + hi)/2
