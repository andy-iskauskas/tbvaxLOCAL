TBVx_parameters_template.csv

The generic input parameters for the country of interest. 

There are 31 parameters to specify for the generic TB model, with TWO that vary by country, and are indicated with "COUNTRY" in the "mean" column. 

The values for these parameters should be set depending on the country of interest.

Notes for specific columns:

choose:
FALSE = parameter does NOT vary during model calibration
TRUE  = parameter DOES vary during model calibration
